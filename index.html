Isis Web

An awesome Enyo-based web browser for webOS.

Download this project as a .zip file Download this project as a tar.gz file

Isis Web is an Enyo(1) browser app for webOS. It's based on the stock web browser of webOS but it boasts more features than the stock web browser.

It uses the Enyo WebView control to display web content which in turn uses BrowserAdapter and BrowserServer to talk to webOS WebKit.
